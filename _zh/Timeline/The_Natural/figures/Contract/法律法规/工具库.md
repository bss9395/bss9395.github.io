# 法律中物权的分类
法律中的物权，指权利人对特定的物拥有支配与流转以及利用的权利。
[自物权/完全物权] 所有权与共有权。比如私人所有权/集体共有权。
[他物权/定限物权] 用益物权与担保物权。前者如土地承包经营权/建设用地使用权/宅基地使用权，后者如抵押权/质权/留置权。
[所有权] 指单个权利人就排他性支配的不动产或者动产，享有占有/使用/收益/处分的权利。
[共有权] 指多个权利人就共同支配的不动产或者动产，享有占有/使用/收益/处分的权利。
[用益物权] 指单个权利人就非本人所支配的不动产或者动产，享有占有/使用/收益的权利。
[担保物权] 指单个债权人为确保债务得到清偿，就债务人所支配的物享有优先受偿的权利。

# 法律中债的担保方式
法律中债的担保方式可划分为以人担保与以物担保，信用保证是典型的以人担保。
[信用保证] 即担保人以个人信用或者信誉，向债权人保证债务人履行债务，若债务人履行债务不善，债权人可要求担保人代为清偿债务。
[抵押担保] 即担保人将财产抵押给债权人，财产不转移而且归担保人所有，若债务人履行债务不善，债权人享有抵押权对该财产的处分。
[质押担保] 即担保人将财产质押给债权人，财产有转移但仍归担保人所有，若债务人履行债务不善，债权人享有质押权对该财产的处分。
[留置担保] 即担保人将财产留置给债权人，财产有转移而且归债权人所有，若债务人履行债务不善，债权人享有留置权对该财产的处分。
[定金担保] 即担保人预先支付债权人定金，定金可以是质押或者留置财产，若债务人履行债务不善，担保人不得请求债权人返还回定金。
[反向担保] 即担保人可以额外要求债务人，为担保人提供反向的各种担保，若债务人履行债务不善，担保人代为清偿债务后可追偿债务。

# 法律中物权的效力
法律中物权至少具有排他效力与优先效力以及追及效力。
[排他效力] 同一物上并存物权的排他占有效力，由强到弱依次为：所有权 > 用益物权 > 担保物权，所有权不得同时存在。
[优先效力] 同一物上并存物权的优先受偿效力，依时间顺序设立在先的物权优先于设立在后的物权，他物权优先于自物权。
[追及效力] 特定的物无论辗转流入任何人手中，物的所有权人都可以追及物的非法占有人进行索取，请求其原路返还回物。

# 法律中的债
法律中的债，又可称为债权债务。
[债的主体] 即债权人与债务人，均为人格平等的法律主体，债权人享有权利，债务人履行义务。
[债的客体] 即债权与债务，债权人有权利请求债务人履行法律的规定，债务人有义务应承债权人履行法律的规定。
[债的内容] 即标的(物)，债权与债务所指定的对象，债务人应当为债权人作出的某种特定行为，通常为给付行为。

# 法律中的合同
法律中的合同，是一种狭义的债。
[合同的主体] 即合同缔约方的债权人与债务人，均为人格平等的法律主体，债权人享有权利，债务人履行义务。
[合同的客体] 即合同中约定的债权与债务，债权人有权利请求债务人履行合同的约定，债务人有义务应承债权人履行合同的约定。
[合同的内容] 即合同中约定的标的(物)，合同中约定所指定的对象，债务人应当为债权人作出的某种特定行为，通常为给付行为。

# 法律中债的保全
[催促保全] 若债权人预期债务人履行债务不善，殃及债权人追偿债务人的债务，债权人可以请求法院，中止或者催促追偿债务人的债务，债务人承担违约责任。
[代位追偿] 若债务人怠于追偿次债务人的债务，殃及债权人追偿债务人的债务，债权人可以己方名义，代位债务人追偿次债务人的债务，次债务人应当履行债务。
[撤销保全] 若债务人有减损其财产的恶意行为，殃及债权人追偿债务人的债务，债权人可以请求法院，对债务人的该恶意行为予以撤销，债务人承担配合义务。

# 法律中债的变更
法律中债的变更，通常指债的内容更改，也包括债的主体变更。
[债务更改] 即债的履行内容亦称标的（物）发生更改。
[债权让予] 即债权人将其所享有的债权转让给第三方。
[债务移转] 即债务人将其所承担的债务移转给第三方。
[债权追加] 即第三方参与享有债权而成为债权人之一。
[债务分担] 即第三方参与承担债务而成为债务人之一。

# 法律中债的终止
[债务无效] 即违反法律规定与非基于真实意思表示的合同以及债务自始无效。比如故意串通移民他国的假婚夫妻。
[条款免责] 即债务人履行合同的债务时，满足免责条款或者不满足任何条款。比如特惠品有额外的恕不退换条款。
[债务清偿] 即债务人遵守与债权人的事先约定，按期适当履行并且清偿债务。比如法院对破产公司进行资产清算。
[债务解除] 即债务人与债权人基于约定事由或者法定事由解除合同以及债务。比如雇主解除与搬砖工的雇佣合同。
[债务抵销] 即债务人与债权人彼此承担债务，可以约定对等抵销相应的债务。比如中美两国互帮互助的友好关系。
[债务免除] 即债务人与债权人协商一致，债权人同意免除债务人履行其债务。比如发达国家免除发展国家的债务。
[标的提存] 即债务人基于正当理由，将标的物提存到特定地点便算解脱债务。比如把在马路边捡到的钱交给警察。
[债的混同] 即债务人与债权人同归为一个主体，以致于债权与债务发生混同。比如债务人继承债权人的遗嘱财产。

# 法律中的不当合同
法律中的不当合同，指合同虽然已经成立，但并不完全具备法律效力。
[违反法律] 即合同约定的内容违反法律的规定或者违背道德伦理，因此无效力。
[恶意串通] 即缔约方为实现有损他方权益的目的而互相恶意串通，因此无效力。
[虚假意思] 即缔约方为掩盖某种不良行为而故意以虚假意思表示，因此无效力。
[欺诈胁迫] 即缔约方为使对方陷入错误而故意实施欺诈胁迫行为，因此无效力。
[乘人之危] 即缔约方利用对方因危迫需要而不得不接受苛刻条款，因此可撤销。
[重大误解] 即缔约方由于己方过错造成重大误解对方的真实意思，因此可撤销。
[显失公平] 即合同约定的内容在客观上明显可察觉有失公平正义，因此可撤销。

# 物的关系变更案例
[案例] 房某将自己所有的房屋一套卖给易某，约定交易费用和办理物权登记手续由买方承担。房某收到房款后，将房子和房证一起交给易某，易某搬入居住，但一直未办理物权变更登记。三年后，房某得知此情况后，以不动产登记的权利人身份诉请法院判决解除该房屋买卖合同，退还房款，返还房屋。易某则以已经交付房款并占有房屋与房证为由，主张自己对该房屋的所有权，要求驳回房某的诉讼请求。

[问题] 不动产登记簿的公信力。
由于现实生活中权利的外观与实质不一致的情形无法避免，因此占有或登记可能出于各种原因，为真实、准确地反映动产或不动产上的物权归属与内容。如果交易的当事人信赖该权利外观而从事物权交易，此后又因权利外观与实质不符合而导致交易无效，势必有害于交易的效率和安全。在交易旺盛、物权变动频繁的现代社会，保护当事人对物权公示方法所展现出来的权利状态的信赖，对于维护交易安全而言十分必要。物权的公信原则指，对信赖令人推测物权存在的外观表象，即登记、占有等公示方法的信赖人，即使在该外观表象并不伴随着实质权利而为空虚存在的情形下，也保护该信赖之原则。
在一般情况下，公示是物权变动的生效要件，不动产物权变动未经登记，不发生效力，动产物权变动未经交付，不发生效力。

[分析] 在本案房某和易某的交易行为中，买卖房屋的合同是债权行为，也就是合同行为，而移转买卖房屋的所有权才是物权行为。原因行为是买卖合同，法律后果是房屋所有权的转移。双方意思表示一致，因而合同的效力没有问题。但是，房某将房屋和房证都交给易某后，易某却没有进行物权变更登记，而不动产的物权变动的公示方法是登记，没有登记就没有发生所有权的转移。尽管易某占有这个房子，也使用三年之久，但是他还是没有取得这栋房子的所有权。房某之所以可以以物权登记簿上的权利人的身份起诉，就是因为法律仍然承认其为该房屋的所有权人。
关于这个案件如何处理，还要看法官如何裁量。由于所有权没有转移，可以让登记的权利人行使所有权，判决解除买卖合同，也可以责令易某进行登记，使易某取得所有权。
笔者同意后一种意见，理由是，双方签订房屋买卖合同是真实意思表示，且易某已经交付房款和交易的标的物，也即房屋并已经居住多年。易某虽然未办理房屋产权过户登记，但已经实际占有其房产证。房某的行为是违反诚信原则的行为，不应得到支持。

# 不当行为的构成要件
法律意义上的行为可划分为作为与不作为，作为即行为人作出特定的行为，不作为即行为人不作出特定的行为。
法律意义上的不当行为，根据侵害程度由弱到强依次为，违约行为 < 侵权行为 < 犯罪行为，违法行为等同于狭义的不当行为。
[行为不当] 作为不当指依事先约定或者法律规定应当不作为而为之，不作为不当指依事先约定或者法律规定应当作为而不为之。
[损害事实] 行为人的行为违法造成他方权益遭受实际损害的客观事实成立。
[因果关系] 行为人的行为违法与他方权益遭受损害事实之间存在因果关系。

# 违约责任的归责原则
违约责任通常无须以违约方在主观上有过错为要件。
法律中的违约责任可划分为预期违约责任与实际违约责任。
[明示违约] 违约方作出明确表示在期限届满前不会履行事先的约定。
[默示违约] 有些迹象表明违约方在期限届满前不会履行事先的约定。
[无能履行] 违约方在期限届满时无能力或者被阻却履行事先的约定。
[拒绝履行] 违约方在期限届满时无正当理由却拒绝履行事先的约定。
[逾期履行] 违约方在期限届满前已作出意向但迟延履行事先的约定。
[部分履行] 违约方在期限届满时已完成部分但欠缺履行事先的约定。
[不当履行] 违约方在履行过程中有实际给付但并不符合事先的约定。

# 侵权责任的归责原则
侵权责任通常需要以违约方在主观上有过错为要件。
法律中的过错心理，包括故意心理与过失心理以及过当心理。
法律中过错心理的潜在危害性由大到小依次为，故意心理 > 过失心理 > 过当心理。
[追求性故意] 行为人明显知道自己行为的不良后果，却追求达到目的以致于该后果必然发生的主观心理。
[放任性故意] 行为人明显知道自己行为的不良后果，却放任事情发展以致于该后果或然发生的主观心理。
[懈怠性过失] 行为人应当预见自己行为的不良后果，但过于自信而轻信能够避免该后果发生的主观心理。
[疏忽性过失] 行为人应当预见自己行为的不良后果，但因为疏忽大意而没有预见该后果发生的主观心理。
[防卫性过当] 行为人出于防卫他人行为的不良后果，而明显越过必要限度以致酿成重大损害的主观心理。
[避险性过当] 行为人出于躲避他人行为的不良后果，而明显越过必要限度以致酿成重大损害的主观心理。


[有过错归责原则] 因加害人主动侵害他方权益，加害人在主观上有过错，因此加害人应当承担侵权责任。
[有过错推定原则] 他方权益受侵害成既定事实，客观推定加害人有过错，因此加害人应当承担侵权责任。
[无过错归责原则] 他方权益受侵害成既定事实，行为人有不可推卸责任，因此行为人应当承担侵权责任。

# 侵权责任的免责事由
[自身过错] 可归责于受害人自身故意放任发生的行为或者过失行为，因此加害人免于承担侵权责任。
[他方过错] 可归责于当事人以外他方故意施加的行为或者过失行为，因此加害人免于承担侵权责任。
[不可抗力] 可归责于当事人遭遇不受其意志所控制的外部强大力量，因此加害人免于承担侵权责任。
[意外事故] 可归责于当事人不可预见或者不可避免的偶然意外事故，因此加害人免于承担侵权责任。
[正当防卫] 可归责于加害人因遭受非法侵害遂采取的正当防卫措施，因此加害人免于承担侵权责任。
[紧急避险] 可归责于受害人为避免更多利益损害而采取的紧急措施，因此加害人免于承担侵权责任。
[职务授权] 可归责于加害人有授权在执行职务时所采取的必要措施，因此加害人免于承担侵权责任。
[自助行为] 可归责于加害人为保护自身合法权益在受助前自我援助，因此加害人免于承担侵权责任。
[自甘风险] 可归责于受害人明知或者容许他人侵害但自愿承担风险，因此加害人免于承担侵权责任。

# 法律中的犯罪主体
犯罪主体在通常语境下为自然人，在罕见语境下也指其他法律主体。
犯罪主体可按照其在（共同的）犯罪活动中的分工与贡献进行分类。
[首要犯] 在（共同的）犯罪活动中发挥领导与组织作用的法律主体。比如黑社会的大哥大。
[主导犯] 在（共同的）犯罪活动中发挥主导与策划作用的法律主体。比如街头混混的头头。
[辅从犯] 在（共同的）犯罪活动中发挥辅助与协调作用的法律主体。比如游手好闲的马仔。
[教唆犯] 在（共同的）犯罪活动中发挥教唆与改造作用的法律主体。比如地下传销的恶棍。
[胁从犯] 在（共同的）犯罪活动中因被胁迫而参与进来的法律主体。比如被洗脑的脑残粉。

# 法律中的犯罪阶段
法律中犯罪的主观因素通常包括犯罪动机与犯罪目的，犯罪动机通常会促使形成犯罪目的。
犯罪阶段可划分为犯罪未完成阶段（特指犯罪未遂）与犯罪已完成阶段（特指犯罪既遂），通常在故意犯罪中会对犯罪阶段进行划分。
法律中对犯罪未完成阶段的量刑程度在原则上总是略微轻于对犯罪已完成阶段的量刑程度。
[犯罪动机] 推动犯罪主体实施其行为的内在动力，一般用于分析对犯罪主体的量刑程度。比如假冒他人专利的动机可能是嫉妒吃醋。
[犯罪目的] 犯罪主体为达成某种结果的心理意向，一般用于分析对犯罪主体的定罪类型。比如假冒他人专利的目的可能是牟取暴利。
[犯罪预备] 犯罪主体在犯罪活动中准备实施犯罪行为但未开始实施的的阶段。比如假冒警察敲诈勒索（预备），但被警察及早发现。
[犯罪中止] 犯罪主体在犯罪活动中实施犯罪行为但中途主动选择放弃的阶段。比如假冒警察敲诈勒索（中止），但却发现得不偿失。
[犯罪未遂] 犯罪主体在犯罪活动中实施犯罪行为但未能达成犯罪目的的阶段。比如假冒警察敲诈勒索（未遂），但不巧碰瓷老司机。
[犯罪既遂] 犯罪主体在犯罪活动中实施犯罪行为且最终达成犯罪目的的阶段。比如假冒警察敲诈勒索（既遂），眨眼间竟一夜暴富。

# 法律中的惩罚种类
法律中的惩罚通常包括强制措施/管制刑/人身刑/财产刑/政治刑/行政处罚。
[强制措施] 对公民的人身自由进行限制。比如盯人联防/电子脚镣/卫星跟踪/通讯窃听/拘留羁押/取保候审/监视居住/限制出境。
[管制刑] 对犯罪者的人身自由进行限制。比如社区矫正/戒毒惩教/强制医疗/有期徒刑/无期徒刑/劳动改造/流放荒野。
[人身刑] 对犯罪者的生命健康造成伤害。比如针刺/鞭笞/廷杖/烹煮/炮烙/电击/剥皮/阉割/鸩毒/活埋/绞缢/枭首/枪毙。
[财产刑] 对犯罪者拥有的财产强行没收。比如上缴巨额罚金/扣押不明资产/冻结银行存款/追缴违法所得/没收所有财产。
[政治刑] 对犯罪者拥有的权利予以剥夺。比如剥夺选举与参政权/剥夺言论与出版权/剥夺游行与示威权/剥夺（军）职衔与职权。
[行政处罚] 对犯罪者在行政上给予处罚。比如谴责道歉/限制消费/警告记过/劝退开除/降薪停职/吊销执照/限令整改/职业禁入/驱逐出境。

# 法律中的惩罚裁量
法律中的惩罚裁量由重到轻依次为，从重处罚 > 相符处罚 > 减轻处罚 > 从轻处罚 > 免除处罚。
[累犯] 犯罪者于刑满释放后在法定期限内又累计多次实施犯罪行为。累犯情形原则上从重处罚。
[并罚] 法院法官将犯罪者所犯数个不同罪名之惩罚合并在一起处罚。并罚情形原则上减轻处罚。
[自首] 犯罪者在实施犯罪行为后主动投案并且如实供述其所犯罪行。自首情形原则上从轻处罚。
[坦白] 犯罪者在实施犯罪行为后被抓捕归案但如实供述其所犯罪行。坦白情形原则上从轻处罚。
[立功] 犯罪者协助司法机关侦破案件或者对社会有其它的重大贡献。立功情形原则上减轻处罚。
[缓刑] 法院法官暂缓执行对犯罪者的惩罚并且附条件地撤销该惩罚。缓刑情形本身即从轻处罚。
[减刑] 法院法官可酌情减轻在服刑期内确有悔改意的犯罪者之刑罚。减刑情形本身即减轻处罚。
[假释] 法院法官可附条件地提前释放在服刑期内表现良好的犯罪者。假释情形本身即减轻处罚。
[赦免] 国家元首在特别情况下宣告免除特定犯罪者之刑罚乃至罪名。赦免情形本身即免除处罚。

# 法律制裁的作用
法律制裁体系通常包括民事制裁/行政制裁/刑事制裁。
[补偿与平衡作用] 通过赔偿受害人的各种损失，可以抚慰受害人的心理失衡，从而维持法律的公平与公正。
[惩罚与教导作用] 通过惩罚加害人的各种过错，可以纠正加害人的错误观念，从而教导社会遵守法律秩序。
[剥夺与遏制作用] 通过剥夺加害人的各项权益，可以使加害人丧失违法能力，从而遏制社会中的违法行为。
[威慑与预防作用] 通过令加害人亲身体验痛苦，可以威慑潜在人群畏惧违法，从而预防社会中的危害发生。

[统一赔偿规则] 所有形式的赔偿应当可统一折算成财产损害赔偿。

# 法律中的可用证据
法律中的证据，指可用于确定或者否定案件事实，按照法定程序收集与验证的材料。
整个司法程序，就是收集与查验可用证据，以此证明案件事实以及定罪量刑的过程。
具体可划分为，取证（收集证据）/举证（提出证据）/质证（审查证据）/认证（认定证据）等过程。
[物证人证] 指在作案时遗留下来的物品与痕迹乃至人身。比如被捉奸在床的狗男女。
[案件口供] 指当事人向司法机关就案件情况所作的陈述。比如对嫌疑人的严刑逼供。
[证人证言] 指作证人向司法机关就自己所知所作的陈述。比如外星人见证者的证言。
[视听资料] 指与案件事实有关的影像与文本等数据资料。比如强奸犯的回忆忏悔录。
[电子数据] 指与案件事实有关的经电子设备传播的记录。比如偷税明星的聊天记录。
[公证证明] 指经过法定的公证机构证明的事实或文书等。比如经公证处公证的遗嘱。
[勘验记录] 指侦查人员对案发现场进行勘验所作的记录。比如检验腐烂尸体的笔录。
[鉴定意见] 指专家运用专门知识就案件情况鉴定的意见。比如对传国玉玺的鉴定书。

# 法律中证据的特征
[真实性] 证据应当是实际存在的事实，而非由主观意识臆造的东西。
[关联性] 证据与案件事实有某种关联，对证明案件事实有实际意义。
[合法性] 证据只能经过法定程序收集，采用非法手段有悖公平正义。

# 法律中证据的规则
[司法令状规则] 指侦查人员未获特定令状许可，不得执行侦查任务。
[合法取证规则] 指司法机关不应当采纳以非法手段收集的有害证据。
[不得强迫规则] 指司法机关不得以非人道手段强迫嫌疑人自证其罪。
[关联证据规则] 指与案件事实并没有实际关联的证据，应予以排除。
[传闻证据规则] 指并非证人亲身耳闻目睹的传闻证据，应适当采纳。
[最佳证据规则] 指若不能获得更好的证据，则采纳现有最佳的证据。
[增补佐证规则] 指若现有证据不足以证明，则必须增补证据来佐证。

# 仲裁
平等主体的公民、法人和其他组织之间发生的合同纠纷和其他财产权益纠纷，可以仲裁。

仲裁在解决民事纠纷方面已经成为一种非常重要的手段。仲裁作为一种相对高效、低成本、非公开、柔性化的民商事纠纷解决的手段，已经成为替代司法解决的最重要、最广泛的方法之一。
仲裁的范围仅限于平等主体的公民、法人和其他组织之间发生的合同纠纷与其他财产权益纠纷，在实践中多数是法人之间的合同纠纷，仲裁的审理以不公开为原则。实行一裁终局制，一旦作出裁决就发生法律效力。一旦当事人双方协议选择仲裁，则就排除司法管辖，当事人不得就该纠纷向人民法院诉讼。

《中华人民共和国仲裁法》【第二条】[可仲裁]
平等主体的公民、法人和其他组织之间发生的合同纠纷和其他财产权益纠纷，可以仲裁。

《中华人民共和国仲裁法》【第三条】[不可仲裁]
下列纠纷不能仲裁：
（一）婚姻、收养、监护、扶养、继承纠纷；
（二）依法应当由行政机关处理的行政争议。

《中华人民共和国仲裁法》【第五条】[一裁定终局]
当事人达成仲裁协议，一方向人民法院起诉的，人民法院不予受理，但仲裁协议无效的除外。

《中华人民共和国仲裁法》【第六条】[无级别管辖]
仲裁委员会应当由当事人协议选定。
仲裁不实行级别管辖和地域管辖。

# 公证
公证存在的问题是，公证人员的现场匿名或假名证据保全的合法性问题。
所谓匿名或假名公证保全，指为了便于对现场的证据资料进行取证保全，公证人员在取证前或取证过程不向现场当事人事先表明其身份或真实身份，而是在取证结束后才向现场行为表明其身份或真实身份，并说明其目的的行为。因为没有表明身份就可能存在因为突然袭击或违法诚信导致程序正义的缺失。在程序正义的一般观念和认识上，突然袭击和非公开性都是违反程序正义的行为。从程序正义的价值观而言，法律行为的实施应当公开的，否则其法律结果的发生就是非正当的。

公证是公证机构根据自然人、法人或者其他组织的申请，依照法定程序对民事法律行为、有法律意义的事实和文书的真实性、合法性予以证明的活动。

《中华人民共和国公证法》【第三十六条】[公证作为证据]
经公证的民事法律行为、有法律意义的事实和文书，应当作为认定事实的根据，但有相反证据足以推翻该项公证的除外。

《中华人民共和国公证法》【第十一条】[公证事项]
根据自然人、法人或者其他组织的申请，公证机构办理下列公证事项：
（一）合同；
（二）继承；
（三）委托、声明、赠与、遗嘱；
（四）财产分割；
（五）招标投标、拍卖；
（六）婚姻状况、亲属关系、收养关系；
（七）出生、生存、死亡、身份、经历、学历、学位、职务、职称、有无违法犯罪
记录；
（八）公司章程；
（九）保全证据；
（十）文书上的签名、印鉴、日期，文书的副本、影印本与原本相符；
（十一）自然人、法人或者其他组织自愿申请办理的其他公证事项。
法律、行政法规规定应当公证的事项，有关自然人、法人或者其他组织应当向公证机构申请办理公证。
